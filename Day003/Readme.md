# Day 4 - Positive, Negative or Zero

## 📌 Problem Statement

Take an integer `n` as input.

Print:

- `Positive` if `n > 0`
- `Negative` if `n < 0`
- `Zero` if `n == 0`

---

## 🧠 Approach

1. Read the integer `n`.
2. If `n > 0`, print **Positive**.
3. Else if `n < 0`, print **Negative**.
4. Otherwise, print **Zero**.

---

## 💻 Example

### Input

```text
5
```

### Output

```text
Positive
```

---

**O(1)**
