# Day 7 - Multiplication Table

## 📌 Problem Statement

Take an integer `n` as input.

Print the multiplication table of `n` from **1 to 10**.

---

## 🧠 Approach

1. Read the integer `n`.
2. Use a `for` loop from `1` to `10`.
3. In each iteration, print `n × i`.

---

## 💻 Example

### Input

```text
5
```

### Output

```text
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
```

---

## ⏱️ Time Complexity

**O(10)** ≈ **O(1)**

---

## 💾 Space Complexity

**O(1)**
