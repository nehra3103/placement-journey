# Day 8 - Count Digits

## 📌 Problem Statement

Take a positive integer `n` as input.

Print the total number of digits in the number.

---

## 🧠 Approach

1. Read the integer `n`.
2. Initialize a counter variable to `0`.
3. Use a `while` loop until `n` becomes `0`.
4. In each iteration:
   - Increase the counter by `1`.
   - Divide `n` by `10`.
5. Print the counter.

---

## 💻 Example

### Input

```text
12345
```

### Output

```text
5
```

---

## ⏱️ Time Complexity

**O(d)**

where `d` is the number of digits.

---

## 💾 Space Complexity

**O(1)**
