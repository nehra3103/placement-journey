# Day 8 - Factorial of a Number

## 📌 Problem Statement

Take a non-negative integer `n` as input.

Print the factorial of `n`.

The factorial of a number is defined as:

`n! = n × (n - 1) × (n - 2) × ... × 1`

By definition,

`0! = 1`

---

## 🧠 Approach

1. Read the integer `n`.
2. Initialize a variable `factorial` to `1`.
3. Use a `for` loop from `1` to `n`.
4. Multiply `factorial` by the current loop variable in each iteration.
5. Print the final value of `factorial`.

---

## 💻 Example

### Input

```text
5
```

### Output

```text
120
```

### Input

```text
0
```

### Output

```text
1
```

---
