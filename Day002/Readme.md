# Day 2 - Even or Odd

## Problem Statement
Take an integer `n` as input.

- Print **"Even"** if `n` is divisible by 2.
- Otherwise print **"Odd"**.

## Approach
1. Read the integer `n`.
2. Check if `n % 2 == 0`.
3. Print `"Even"` if true, otherwise print `"Odd"`.

## Time Complexity
O(1)

## Space Complexity
O(1)
