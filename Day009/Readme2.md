# Day 16 - Greatest of Three Numbers

## 📌 Problem Statement

Take three integers `a`, `b`, and `c` as input.

Print the greatest among them.

---

## 🧠 Approach

1. Read the three integers `a`, `b`, and `c`.
2. Compare the numbers using `if-else` statements.
3. If `a` is greater than or equal to both `b` and `c`, print `a`.
4. Otherwise, if `b` is greater than or equal to both `a` and `c`, print `b`.
5. Otherwise, print `c`.

---

## 💻 Example

### Input

```text
10 25 18
```

### Output

```text
25
```

### Input

```text
7 7 3
```

### Output

```text
7
```
